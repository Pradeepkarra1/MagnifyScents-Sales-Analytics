## 📌 Project Overview

End-to-end **Sales & Marketing Analytics** project for **MagnifyScents**, a perfume/fragrance brand. This project covers exploratory data analysis (EDA), revenue trend analysis, customer segmentation, product performance, and KPI dashboards — transforming raw transactional data into actionable business insights.

---

## 🎯 Business Questions Answered

- Which products generate the highest revenue and profit margins?
- What are the monthly and seasonal sales trends?
- Which customer segments drive the most lifetime value?
- Which marketing channels have the best ROI?
- What pricing strategies can maximize revenue?

```bash

# Install dependencies
pip install -r requirements.txt
```
