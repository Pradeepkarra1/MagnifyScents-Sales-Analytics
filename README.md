## 📌 Project Overview

End-to-end **Sales & Marketing Analytics** project for **MagnifyScents**, a perfume/fragrance brand, built entirely in **Microsoft Excel**. This project uses Pivot Tables, Power Query, Excel formulas, charts, and KPI dashboards to transform raw transactional data into clear, actionable business insights — **no coding required**.

### What This Project Demonstrates:
- ✅ Real-world business analytics on a consumer brand dataset
- ✅ Advanced Excel skills: Power Query, Pivot Tables, Slicers, Dynamic Charts
- ✅ Customer segmentation and cohort analysis
- ✅ Revenue trend analysis and KPI tracking
- ✅ Data-driven strategic recommendations for business growth

## 📊 Key Insights & KPIs

| Metric | Finding |
|--------|---------|
| 📈 Top Product Concentration | Top 3 SKUs = ~60% of total revenue |
| 🗓️ Peak Season | Oct–Dec drives highest sales (festive surge) |
| 👤 Pareto Principle | Top 20% customers = ~80% of GMV |
| 📉 Discount Impact | Deep Q2 discounts reduced margins by ~12% |
| 🔁 Repeat Rate | ~35–40% of customers return within 90 days |
| 💰 Best Channel | Direct website = highest AOV and best margins |
| 🛒 Bundle Effect | Bundle sets deliver 25% higher AOV vs. single SKUs |

## 🚀 How to Use This Project

1. **Download** `MagnifyScents_Final.xlsx` from the [`data/`](data/) folder
2. Open in **Microsoft Excel 2016 or later** (or Microsoft 365)
3. Click **Enable Editing** and **Enable Content** if prompted
4. Navigate using the **sheet tabs** at the bottom
5. Use **Slicers and Timelines** on dashboard sheets to filter interactively
6. To refresh data connections: `Data` → `Refresh All`

> ⚠️ **Best viewed in Excel for Windows (Microsoft 365).** Some features may be limited in Excel for Mac or Google Sheets.

## 💾 Data Description

## 💡 Top Strategic Recommendations

1. 🏆 **Launch VIP Loyalty Program** — target top 20% customers with exclusive access and personalized offers
2. 📦 **Q2 Bundle Campaigns** — replace deep discounts with curated bundles to protect margins
3. 🌐 **Grow Direct Channel** — reduce marketplace dependency through SEO and referral programs
4. ✂️ **Cap Discounts at 15%** — data shows no volume uplift beyond this threshold
5. 📲 **30-Day Re-engagement Flow** — automated post-purchase touchpoint to convert one-time buyers
