## 📌 Project Overview

End-to-end **Sales & Marketing Analytics** project for **MagnifyScents**, a perfume/fragrance brand, built entirely in **Microsoft Excel**. This project uses Pivot Tables, Power Query, Excel formulas, charts, and KPI dashboards to transform raw transactional data into clear, actionable business insights — no coding required.

---

## 🎯 Business Questions Answered

- Which products generate the highest revenue and profit margins?
- What are the monthly and seasonal sales trends?
- Which customer segments drive the most repeat purchases?
- Which channels and promotions deliver the best ROI?
- Where are the pricing and discount inefficiencies?

## 🔧 Tools & Excel Features Used

| Feature | Purpose |
|---------|---------|
| Pivot Tables | Sales aggregation by product, region, channel |
| Power Query | Data cleaning, merging, and transformation |
| Pivot Charts | Visual trend and comparison charts |

## 📊 Key Insights & KPIs

- 📈 **Top 3 products** account for **~60% of total revenue**
- 🗓️ **Peak sales months**: October–December (festive season surge)
- 👤 **High-value customers** (top 20%) contribute **~80% of GMV** (Pareto principle)
- 💰 **Average Order Value (AOV)** tracked across all channels
- 📉 **Heavy discounting in Q2** reduced margins by ~12%
- 🔁 **Repeat purchase patterns** identified through customer cohort analysis

## 🚀 How to Use

1. Download **`MagnifyScents_Final.xlsx`** from the `data/` folder
2. Open in **Microsoft Excel 2016 or later** (or Microsoft 365)
3. Enable editing and content (macros/Power Query connections)
4. Use the **slicers and timeline** on the dashboard sheets to filter by date, product, or channel
5. Refresh Power Query connections if updating with new data (`Data → Refresh All`)

> **Note:** Works best in Excel for Windows. Some features may be limited in Excel for Mac or Google Sheets.
